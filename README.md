## Descripción del taller

Este taller tiene 13 ejercicios que van desde cosas básicas como calcular factoriales hasta funciones un poco más complejas como la función de Ackerman. La idea es practicar ciclos, recursividad y manejo de arreglos y matrices.

---

## Estructura del proyecto

```
src/
├── ejercicioUno/        # Factorial de un número
├── ejercicioDos/        # Sumatoria hasta N
├── ejercicioTres/       # Sumatoria 1 + 1/2 + 1/3 + ... + 1/n
├── ejercicioCuatro/     # Invertir un número
├── ejercicioCinco/      # Suma de dígitos
├── ejercicioSeis/      # Potencia (base^exponente)
├── ejercicioSiete/       # Máximo Común Divisor (Algoritmo de Euclides)
├── ejercicioOcho/      # Cociente por restas sucesivas
├── ejercicioNueve/       # Multiplicación por sumas sucesivas
├── ejercicioDiez/       # Suma de elementos de un vector
├── ejercicioOnce/       # Suma de elementos de una matriz mxn
├── ejercicioDoce/      # Serie de Fibonacci
└── ejercicioTrece/    # Función de Ackerman
```

## Requisitos previos

Para ejecutar este proyecto se necesita tener instalado lo siguiente en tu máquina. El **JDK 11 o superior** es indispensable para compilar y ejecutar código Java. El **IDE recomendado es Visual Studio Code** con la extensión *Extension Pack for Java* de Microsoft, aunque también puedes usar IntelliJ IDEA o NetBeans.

Puedes verificar que tienes Java instalado ejecutando este comando en tu terminal:

```bash
java -version
```

Si el comando no es reconocido, descarga el JDK desde [https://www.oracle.com/middleeast/java/technologies/downloads/]

---

## Instrucciones para ejecutar el programa

**Desde Visual Studio Code**

1. Abre la carpeta raíz `Taller-Recursividad` en VS Code con `Archivo > Abrir carpeta`.
3. Navega hasta el archivo `App.java` del ejercicio que quieras ejecutar dentro de `src/`.
4. Haz clic en el botón **▷ Run** que aparece encima del método `main`, o presiona `F5`.
5. El resultado se mostrará en el panel **Terminal** en la parte inferior de la pantalla.
